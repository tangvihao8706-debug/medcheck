# MedCheck - Healthcare Web App

## 📌 Description
MedCheck is a simple healthcare web application that provides user interaction and basic data display.

## 🎯 Purpose
This project was developed to practice frontend development and improve UI/UX design skills.

## 🚀 Features
- User interface for healthcare-related interaction
- Responsive layout
- Basic data display

## 🛠 Tech Stack
- HTML, CSS, JavaScript

## ▶️ How to run
1. Download or clone the repository
2. Open index.html in your browser
