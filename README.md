# MedCheck - Healthcare Web Application (Frontend)

## 📌 Description
MedCheck is a frontend healthcare web application built using HTML, CSS, and JavaScript.
The application focuses on providing a simple and user-friendly interface for displaying healthcare-related information.

## 🎯 Purpose
This project was developed to practice frontend development, improve UI/UX design skills, and understand how to structure a web interface.

## 🚀 Features
-Clean and user-friendly interface
-Responsive layout for different screen sizes
-Display of healthcare-related information
-Basic user interaction with UI elements
## 🛠 Tech Stack
- HTML
- CSS
- JavaScript

## ▶️ How to run
1. Download or clone the repository
2. Open index.html in your browser
